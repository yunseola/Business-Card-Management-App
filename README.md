# 📇 명함.zip (Business Card Management App)

**명함.zip**은 종이 명함의 한계를 해결하고, 쉽고 편리하게 명함을 관리할 수 있도록 만든 모바일 앱입니다.  
종이 명함의 내구성 문제와 수정 불가능성, 환경적 문제를 해결하며, 스마트폰 하나로 명함을 관리하고 공유할 수 있습니다.

---

## ✨ 주요 기능
- **전화 수신 시 명함 위젯**  
  → 연락처에 저장된 사람이 전화를 걸면, 앱에 등록된 명함 정보와 사용자가 작성한 메모가 화면에 표시됩니다.  
  → 여러 메모가 있을 경우 **AI 요약 기능(OpenAI API)** 이 자동으로 핵심 한 줄 메모를 추출하여 빠른 대응이 가능하도록 지원합니다.

- **명함 공유**  
  → **QR 코드** 또는 **링크 공유** 기능을 통해 언제 어디서나 명함을 전달할 수 있습니다.  
  → 종이 명함이 없는 상황에서도 스마트폰만 있으면 간편하게 명함 교환이 가능합니다.  
  → 흔들기(Gesture) 기능으로 가까운 사용자와 명함을 바로 교환할 수 있습니다.

- **명함 생성 및 관리**  
  → 앱 내에서 내 명함을 직접 제작할 수 있고, 다양한 디자인 옵션을 제공합니다.  
  → 받은 명함을 카테고리별로 정리/검색할 수 있어 체계적인 관리가 가능합니다.  
  → 종이 명함은 카메라 촬영 후 **OCR(ML Kit + Clova OCR)** 을 통해 자동으로 텍스트를 인식하고 저장할 수 있습니다.

- **메모 등록 및 관리**  
  → 명함별 맞춤 메모를 작성할 수 있고, 해당 메모는 전화 수신 시 함께 표시됩니다.  
  → 중요한 대화 포인트를 기록해두면 추후 다시 연락할 때 도움이 됩니다.

---

## 🛠 사용 기술
- **Frontend & Mobile**  
  - Android Studio, Kotlin 기반 UI 구현  
  - CameraX 라이브러리를 통한 카메라 촬영 및 명함 스캔 기능  

- **OCR & AI**  
  - **Google ML Kit, Naver Clova OCR** → 명함 이미지에서 텍스트 추출  
  - **OpenAI GPT API** → 추출된 텍스트를 이름/전화번호/이메일/직책 등 필드별로 자동 분류 + 메모 요약  

- **Backend & Data**  
  - Spring Boot 기반 API 서버와 연동  
  - Google OAuth & MySQL → 사용자 인증 및 데이터 관리  
  - RESTful API를 통한 명함 CRUD(Create, Read, Update, Delete) 기능 구현  

- **협업 & 관리**  
  - GitLab, Jira를 통한 버전 관리 및 일정 관리  
  - Figma로 UI/UX 디자인 및 프로토타입 제작
